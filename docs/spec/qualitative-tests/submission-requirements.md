# Submission requirements

## 1. High level of artifact

- [ ] submisson is a zip file
- [ ] submission contains a cover document, which is a separate artifact
from the readme.

## 2. Cover document

- [ ] zipped archive contains the group name AND project url.
- [ ] contains a breakdown of each members contributions.
- [ ] the breakdown includes an estimated percentage of contributon
- [ ] Specific features that people focused on

## 3. AI usage report

- [ ] Screenshots _OR_ exported logs of significant prompts and responses
- [ ] A brief reflective commentary of:
    - [ ] how AI tools were used
    - [ ] what worked well
    - [ ] what required manual correction
- [ ] Evidence of critical evaluation of AI-generated code.
